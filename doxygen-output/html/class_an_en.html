errorType AnEn::computeStandardDeviation ( const Forecasts_array &  forecasts,
StandardDeviation &  sds 
)

Computes the standard deviation of times of forecasts for each parameter, each station, and each FLT. This function is designed to generate NAN values only when all values for a parameter, a station, and a FLT are NAN. Otherwise, NAN values will be ignored.

See StandardDeviation for more information about storage.

Parameters
forecastsForecasts for which standard deviation is generated for.
sdsStandardDeviation to store the values.
